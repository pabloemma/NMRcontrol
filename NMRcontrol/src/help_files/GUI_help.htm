The NMR analyzer GUI

 

This is the help file for the NMR GUI

 

 

This is the current main interface to the NMR analyzer. In order to run the analyzer, a few things have to be  defined.

1.     The parameter file. This is where all the parameters are store. The current set is shown in the table and explained in the NMR analyzer help documentation.

2.     Input files: Click on choose and it first lets you get to the right directory. Once you are there you can select as many files as you want. Those files will show up in “input files window”

3.     once you are done , hit run and it should start working. Currently the stop button does not do anything since I do not want to exit with a kill
